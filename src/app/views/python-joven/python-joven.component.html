python_joven

#uruguay 2009 - 2014

Descripción:

Comunidad uruguaya de jóvenes entusiastas de la programación, unidos en torno a python y el software libre, buscando llegar lo más lejos posible en el aprendizaje sobre el desarrollo de software.
Compartimos el conocimiento adquirido brindando oportunidades de aprendizaje a aquellos jóvenes que estén dispuestos a dedicar su tiempo a esta tarea.
