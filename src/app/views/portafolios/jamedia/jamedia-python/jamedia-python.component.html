JAMedia en python

  • Reproduce Archivos de Audio y Video
  • Reproduce Streamings
  • Busca Videos en youtube
  • Descarga Videos desde youtube
  • Graba desde Streamings
Repositorio: jamedia-python
Instalador para Sugar: jamedia-sugar

Bibliotecas Necesarias:

  • python 2.7
  • gtk 2.0
  • gstreamer 0.10
  • python-gst0.10
  • gstreamer0.10-ffmpeg
  • gstreamer0.10-plugins-base
  • gstreamer0.10-plugins-good
  • gstreamer0.10-plugins-ugly
  • gstreamer0.10-plugins-bad
  • gstreamer0.10-tools
  • python-gst0.10-rtsp

Guia de Instalación:

Descarga:
  • Descarga el zip desde el repositorio
  • Descomprime el zip en tu computadora
  • Entra en el directorio descomprimido
Instalación standar:
  • Ejecuta: "sudo python setup.py install"
Instalación sin root:
  • Ejecuta: "python install.py"

Para Desinstalar:

Instalación standar:
  • Ejecuta: "sudo jamedia_uninstall"
Instalación sin root:
  • Borra: "/home/tu_usuario/.local/share/JAMedia"
  • Borra: "/home/tu_usuario/.local/share/applications/JAMedia.desktop"
