JAMedia en python

  • Reproductor de Archivos de Audio y Video.
  • Reproductor de Streamings de radios.
  • Buscador masivo de videos en youtube.
  • Descarga masiva de videos desde youtube.
  • Extractor masivo de audio e imágenes desde archivos de videos.
  • Conversor masivo de archivos de audio y video a diversos formatos.
Repositorio: jamedia-python

Bibliotecas Necesarias:

  • python 3.6.3
  • gtk 3.0
  • gstreamer 1.0
  • webkit 2-4.0
  • gstreamer1.0
  • gstreamer1.0-plugins-base
  • gstreamer1.0-plugins-good
  • gstreamer1.0-plugins-bad
  • gstreamer1.0-plugins-ugly
  • gstreamer1.0-libav
  • gstreamer1.0-tools
  • gir1.2-webkit2-4.0

Guia de Instalación:

Instala las librerías:
  • En la terminal Ejecuta: sudo apt-get install gstreamer1.0 gstreamer1.0-plugins-base gstreamer1.0-plugins-good gstreamer1.0-plugins-bad gstreamer1.0-plugins-ugly gstreamer1.0-libav gstreamer1.0-tools gir1.2-webkit2-4.0
Descarga JAMedia:
  • Descarga el archivo deb desde el repositorio
Instalación:
  • Hazle doble click al archivo descargado

Para Desinstalar:

  • Ejecuta: "sudo apt-get remove jamedia"
