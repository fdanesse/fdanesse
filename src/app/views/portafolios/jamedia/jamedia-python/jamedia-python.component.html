JAMedia en python

  • Reproduce Archivos de Audio y Video
  • Reproduce Streamings
  • Busca Videos en youtube
  • Descarga Videos desde youtube
  • Graba desde Streamings (Quitado en Versión 18)
Repositorio: jamedia-python
Instalador para Sugar: jamedia-sugar

Bibliotecas Necesarias:

  • python 3.6.3
  • gtk 3.0
  • gstreamer 1.0
  • webkit 4.0
  • gstreamer1.0-plugins-base
  • gstreamer1.0-plugins-good
  • gstreamer1.0-plugins-ugly
  • gstreamer1.0-libav
  • gstreamer1.0-tools

Guia de Instalación:

Descarga:
  • Descarga el archivo deb desde el repositorio
Instalación:
  • Hazle doble click al archivo descargado

Para Desinstalar:

  • Ejecuta: "sudo apt-get remove jamedia"
